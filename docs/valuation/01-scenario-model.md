# 情景模型

## 结论

CRCL 情景模型分为 Bull、Base、Bear 三类。

情景不是价格预测。
情景用于判断哪些数据会让研究框架升级或降级。

## Bull case

触发条件：

1. USDC 30D change 持续为正。
2. USDC 市占率连续 4 周上升。
3. RLDC margin 稳在 40% 以上。
4. Other revenue 超过 FY2026 指引上沿。
5. CPN TPV 与收入同步增长。
6. Arc 主网按期落地，并披露真实客户和费用模型。
7. 监管最终规则没有实质压制分销激励。
8. competition score 高于 75。
9. Base、Solana、Ethereum 增长同时出现，且不是单一渠道内循环。

结论：

CRCL 具备从利差股向合规支付基础设施股重定价的条件。

动作：

提高平台化权重，但继续用 RLDC 验证基础盘。

## Base case

触发条件：

1. USDC 随稳定币总盘增长，市占率横盘。
2. 储备收益率下行，但流通量增长抵消部分压力。
3. RLDC margin 维持 38-40%。
4. Other revenue 达到 FY2026 1.5-1.7 亿美元指引。
5. CPN 和 Arc 有进展，但收入贡献仍不清晰。
6. competition score 维持 55-74。
7. 收益型稳定币和 RWA 增长没有触发 USDC 净赎回。

结论：

CRCL 仍主要按利差股研究，平台化给期权价值。

动作：

维持基础框架，不因单一产品公告上调估值。

## Bear case

触发条件：

1. USDC 连续 2-4 周净赎回。
2. USDC 市占率被 USDT、银行稳定币或收益型稳定币挤压。
3. RLDC margin 下滑，渠道拿走更多经济性。
4. OCC / Treasury 最终规则限制交易所或第三方激励。
5. CPN TPV 增长但 Other revenue 不增长。
6. Arc 主网延期或费用模型无商业价值。
7. competition score 低于 40。
8. Base USDC 增长但总 USDC 份额不升，且 RLDC margin 下滑。
9. 银行稳定币或 tokenized deposits 获得企业结算采用。

结论：

CRCL 从平台化叙事回落为高波动利差股，倍数需要下修。

动作：

暂停加仓，重算收入、成本和估值倍数。

| 当前情景 | 转向更强                                                          | 转向更弱                                                         |
| -------- | ----------------------------------------------------------------- | ---------------------------------------------------------------- |
| Bear     | USDC 30D 转正，市占率企稳，RLDC margin 修复                       | 储备信用异常，监管压制分销                                       |

| Bull     | 平台化收入连续增长，监管清晰，市占率上升                          | 收入不及预期，Base 增长但利润留存恶化（筹码压力不触发情景切换，仅影响仓位操作） |

## Autoresearch 情景门槛

情景切换必须满足矩阵共振，不能由单一指标触发。

| 目标情景 | 必须满足的矩阵结果                         | 禁止切换条件                         |
| -------- | ------------------------------------------ | ------------------------------------ |
| Bull     | 财务强化、竞争强化、平台化中验证以上、监管无 P0 | RLDC margin 下滑或 Other revenue 未拆分 |
| Base     | 财务中性、竞争中性、平台化弱验证、监管未落地 | 信用风险或连续净赎回                 |
| Bear     | 财务降级、竞争恶化、监管压制、平台化失败任意两项 | 单一筹码波动且基本面未变             |

中验证是平台化中等验证，白话就是 Other revenue 占比超过 10% 且连续两个季度增长。

## 操作执行层说明（独立于情景判断）

以下筹码信号（P2 指标）触发仓位上限下调，独立于情景切换，不参与情景判断：

- 大股东或创始人单季度内减持 > 5%
- 做空比率超过流通股 15%
- 解禁、增发、内部人 Form 4 集中出现

触发后执行 `metrics/00-metric-dictionary.md` P2 指标中的筹码风控流程，不切换情景框架。

筹码压力是短期波动放大器，不是商业模式变化信号。单独筹码波动不修改估值倍数，不触发情景切换。

| 情景 | 主网时间线 | 收入逻辑 |
| ---- | ---------- | -------- |
| Bear | 主网延期，无明确新时间表 | 0 收入，Arc 期权价值暂停计算 |

### 企业 API / 集成服务假设

FY2025 Other revenue = 1.098 亿，其中 CPN 早期贡献估算约 0.05 亿（基于年化 57 亿 TPV × 约 0.08% 隐含费率），Arc 约 0（测试网），企业 API / 集成约 1.05 亿。

Bear 假设（0.77 亿）：平台化失速，新客户获取放缓，部分一次性集成收入不可重复。
Base 假设（1.30 亿）：稳步增长，新企业客户持续接入，收入可重复性提升。
Bull 假设（1.40 亿）：CPN / Arc 生态带动 API 使用，企业客户质量改善，但部分收入重分类至 CPN / Arc 子项。

## Base USDC 增速假设细化 **[补充]**

USDC 平均流通量假设基于稳定币总盘增速和 USDC 市占率变化两个维度。

### 稳定币总盘与 USDC 市占率假设

| 维度 | Bear | Base | Bull |
| ---- | ---- | ---- | ---- |
| 稳定币总盘 FY2026 YoY | +10% | +20-25% | +35-40% |
| USDC 市占率变化（ppts） | -2 至 -3 | ±1 | +2 至 +3 |
| 参数表平均 USDC 流通量（亿美元） | 700 | 850 | 1,050 |
| 对应隐含 USDC YoY（vs Q4 2025 末 753 亿） | 约 -7% | 约 +13% | 约 +39% |

参考基准：Q4 2025 末 USDC = 753 亿，Q4 2025 平均 USDC = 762 亿，稳定币总市值估算约 2,300 亿，USDC 占比约 32-33%（来源：`framework/01-business-model.md`）。

### USDC 市占率路径分析

**Bear 路径（丧失 2-3ppts）**

- USDT 加速扩张（企业和机构采用），绝对增量高于 USDC。
- 银行稳定币获得企业结算采用，切走 USDC 部分份额。
- 监管限制激励，分销渠道议价倒向 USDT。
- USDe / RWA 对收益敏感型资金产生替代，USDC 净赎回同步出现。

**Base 路径（横盘 ±1ppt）**

- USDC 随稳定币总盘有机增长，受益于合规化叙事和 Coinbase 生态。
- CPN 早期采用补充机构需求，维持基本盘。
- 竞争压力与合规优势基本抵消，市占率波动但无趋势性下降。

**Bull 路径（扩大 2-3ppts）**

- GENIUS Act 等监管立法赋予 USDC 合规背书，机构客户偏好 USDC。
- CPN 大规模企业采用，B2B 跨境支付替代效应显现。
- Arc 主网上线带动链上经济 USDC 需求（DeFi 和企业结算）。
- 竞争对手（银行稳定币、USDT）在合规受监管市场受限，USDC 获益。

### 计算示例（Base）

```text
Q4 2025 末 USDC = 753 亿，稳定币总盘约 2,300 亿，USDC 占比约 32.7%
FY2026 Base：稳定币总盘增长 +22% → 约 2,806 亿
USDC 市占率维持 32-33% → FY2026 末 USDC ≈ 925-940 亿
平均 USDC ≈（753 + 930）÷ 2 ≈ 842 亿 ≈ 参数表 850 亿（合理）
```

## 利率敏感度矩阵 **[补充]**

### 基准参数（Q4 2025 实际值）

| 参数 | 历史基准值 | 来源 |
| ---- | ------ | ---- |
| 5Y US Treasury | 约 4.3% | FRED / Bloomberg |
| Circle Reserve Fund（USDXX）7 日收益率 | 约 3.8% | BlackRock 基金页面 |
| Spread（管理费 / 托管费等） | 约 50bps | 隐含值（4.3% - 3.8%） |
| FY2026 Base 平均 USDC 流通量 | 850 亿美元 | 参数表（本文件） |
| FY2025 RLDC margin（主口径） | 约 41% | `framework/01-business-model.md` |

### 降息传导机制

```text
联储降息
  → 3M T-Bill 几乎即时响应（~1 个月内 95% 传导）
  → USDXX 基金：以短期国债和回购为主，~1-3 个月充分传导
  → Reserve return rate 响应：约 85% 传导系数（久期缓冲效应）
  → Spread（约 50bps）维持相对稳定（管理费和托管费固定成本）
```

WAM（加权平均久期）通常 30-60 天，快速降息时储备收益率下行可能滞后 1-2 个月，快速加息时也有类似滞后。

### 降息敏感度矩阵（基于 Base 情景 850 亿 USDC）

| 联储降息幅度 | 5Y Treasury | 储备收益率（USDXX） | 年化毛储备收入 | 年化 RLDC（41%） | vs 不降息 ∆RLDC |
| ------------ | ----------- | ------------------- | -------------- | --------------- | --------------- |
| 不降息（基准） | 4.3% | 3.8% | 850×3.8%=32.3 亿 | 13.2 亿 | — |
| 降息 50bps | ~3.8% | ~3.4% | 850×3.4%=28.9 亿 | 11.9 亿 | **-1.4 亿** |
| 降息 100bps | ~3.3% | ~3.0% | 850×3.0%=25.5 亿 | 10.5 亿 | **-2.8 亿** |
| 降息 150bps | ~2.8% | ~2.5% | 850×2.5%=21.3 亿 | 8.7 亿 | **-4.5 亿** |

储备收益率传导系数假设约 85%。实际传导取决于 USDXX 当期 WAM，需从 BlackRock 基金月报更新。

### RLDC 警戒线分析

```text
38% 告警线（主口径）在各降息情景下对应的 RLDC：

  不降息：850 × 3.8% × 38% = 12.3 亿（低于 41% 基准 13.2 亿，差距 0.9 亿）
  降息 50bp：850 × 3.4% × 38% = 11.0 亿
  降息 100bp：850 × 3.0% × 38% = 9.7 亿
  降息 150bp：850 × 2.5% × 38% = 8.1 亿

当联储降息 ≥100bps 且 RLDC margin 同步下滑至 38% 时，
RLDC 将跌至 9.7 亿，低于参数表 Bear 情景（~8.5 亿）约 14%。
这是"利率下行 + 渠道成本上升"双杀场景，触发 Bear 情景切换条件。
```

### 以规模增长弥补利率下行所需 USDC 量（RLDC margin 维持 41%）

| 降息幅度 | 储备收益率 | 维持 RLDC 13.2 亿所需 USDC | 相比基准新增 | 可行性 |
| -------- | ---------- | -------------------------- | ------------ | ------ |
| 降息 50bps | 3.4% | 946 亿 | +96 亿（+11%） | 较可行（Base 增速范围内） |
| 降息 100bps | 3.0% | 1,073 亿 | +223 亿（+26%） | 难度较大（接近 Bull 情景） |
| 降息 150bps | 2.5% | 1,288 亿 | +438 亿（+52%） | 极难（超出 Bull 情景上限） |

### 使用规则

1. 每次美联储 FOMC 决议后，用实际降息幅度更新储备收益率假设（以 USDXX 7 日收益率为代理，日频监控）。
2. 降息超预期（单次 > 50bps）触发情景参数表更新和仓位减仓判断（-10ppts，参见估值框架仓位因子表）。
3. 储备收益率下行不自动触发情景降级，需同步确认 USDC 流通量增速能否在合理范围内弥补。
4. 参数表中 Bear 情景已预设储备收益率 3.2%，对应联储较历史基准累计降息约 60bps（从 3.8% 降至 3.2%）；Bull 情景 4.0% 对应联储不降息甚至小幅加息。运行时需用最新 USDXX 7 日收益率重新定位基准。

## 稀释股数锁定记录 **[补充]**

稀释股数用于将目标市值转换为每股目标价。应每季度从最新 10-Q 更新，不能使用超过 1 个季度前的数据。

### 运行时记录

| 项目 | 数值 | 记录日期 | 来源 | 下次更新 |
| ---- | ---- | -------- | ---- | -------- |
| 基本股数（Basic shares outstanding） | [运行时从最新 filing 更新] | [待更新] | 10-Q / 10-K 封面页 "shares outstanding" | 最新财报 filing 后 |
| 完全稀释股数（Diluted shares） | [运行时从最新 filing 更新] | [待更新] | 10-Q / 10-K EPS 注释 "weighted average diluted shares" | 最新财报 filing 后 |
| 期权 / RSU / ESPP 稀释量 | [运行时从最新 filing 更新] | [待更新] | 10-Q / 10-K 股权补偿注释 | 最新财报 filing 后 |
| 潜在增发（shelf offering） | [待确认] | [待更新] | SEC S-3 / 424B 文件 | 事件驱动 |

### 查找方法

```text
1. SEC EDGAR → 搜索 "CRCL" → 最新 10-Q
   → 封面页 "shares outstanding as of [日期]"（基本股数）

2. 10-Q EPS 计算注释
   → "weighted average diluted shares used in computing diluted net income per share"
   → 这是目标价计算应使用的数值

3. 股权激励计划注释（stock-based compensation）
   → 期权授予量、RSU 未行权数、行权价格区间
   → 判断潜在稀释幅度
```

### 更新规则

- 每季度财报后（Q1-Q4）必须更新，不使用超过 1 个季度前的数据。
- 若有增发公告（S-3 / 424B）或重大股权激励发放，事件驱动立即更新。
- 目标价计算必须使用 diluted shares，不用基本股数。
- 若稀释股数较基本股数高 > 3%，检查 stock-based compensation 对 Adjusted EBITDA 的影响。
- 若内部人持股比例在连续两个季度下降 > 2ppts，检查 Form 4 报告确认是否为有计划卖出（10b5-1 plan）还是非预设减持。

### 目标价计算提示

```text
目标每股价 = 目标市值 ÷ 稀释股数

示例（Base case 中性）：
  年化 RLDC ≈ 12.6 亿美元
  中性倍数：18×
  目标市值 = 12.6 × 18 = 226.8 亿美元
  若稀释股数 = X 亿股
  目标每股价 = 226.8 ÷ X 美元

稀释股数应从最新 10-Q 获取，每季度更新，不能从旧文件线性外推。
```
