# 竞争格局

## 结论

CRCL 的竞争不是单一 USDT 对抗。

竞争分为五类：

1. USDT：规模、离岸流动性、交易所深度。
2. Coinbase：渠道伙伴和议价对手。
3. 银行稳定币与 tokenized deposits：制度竞争。
4. 收益型稳定币与 RWA 产品：资金收益竞争。
5. 支付网络与企业结算方案：长期场景竞争。

tokenized deposits 是代币化存款，白话就是银行把存款搬到链上表达。
RWA 是 Real World Assets，白话就是美债、货币基金等现实资产被搬到链上。

## USDT / Tether

USDT 的优势：

- 最大稳定币网络效应。
- 交易所深度强。
- 离岸市场使用惯性强。
- 盈利能力强，能用利润防守。

USDT 的弱点：

- 美国合规路径不如 Circle 清晰。
- 储备透明度和监管折价长期存在。
- 美国机构和企业支付场景接受度受限。

对 CRCL 的判断：

USDC 不需要在所有场景复制 USDT。
CRCL 的胜负手是合规美元结算网络，而不是离岸交易稳定币第一名。

## Coinbase

Coinbase 对 CRCL 同时有三种身份：

1. 分销入口：带来用户、钱包、Base 链流动性。
2. 成本中心：分成会压缩 RLDC margin。
3. 议价对手：Base 和交易所掌握终端用户关系。

Base 是 Coinbase 推出的 Layer-2 网络，白话就是 Coinbase 控制入口的链上生态。
Layer-2 是二层网络，白话就是建在主链上的低成本交易网络。

关键判断：

```text
Base 链 USDC 增长
  如果 RLDC margin 同步稳定或上升 = 高质量增长
  如果 RLDC margin 下滑 = 渠道拿走更多经济性
```

## 银行稳定币与 tokenized deposits

银行稳定币的优势：

- 监管关系更近。
- 企业客户基础强。
- 存款、托管、支付链路一体化。

银行稳定币的劣势：

- 跨链开放性弱。
- 产品迭代和开发者生态弱。
- 多银行之间互操作性需要时间。

对 CRCL 的影响：

银行不会先抢走 crypto 交易场景。
银行会先影响企业结算、机构支付、受监管钱包和 B2B 网络的估值想象。

B2B 是 Business-to-Business，白话就是企业对企业。

## 收益型稳定币与 RWA

收益型竞品包括 USDe、sUSDe、BUIDL、USDY、USYC 等。

它们竞争的是闲置美元收益，而不是所有支付场景。

关键判断：

| 情形                     | 对 CRCL 影响                         |
| ------------------------ | ------------------------------------ |
| 风险偏好强，链上杠杆上升 | USDC 使用量增长，收益型产品也增长    |
| 短端利率高，用户追收益   | 闲置 USDC 被收益型产品分流           |
| 监管限制收益分享         | 合规稳定币更难直接拼收益             |
| 企业支付增长             | 收益型产品影响下降，结算稳定性更重要 |

## 护城河评分

| 因素       | 强                      | 弱               |
| ---------- | ----------------------- | ---------------- |
| 合规路径   | 许可清晰、披露可信      | 监管不确定       |
| 储备透明   | 官方披露稳定            | 报告延迟或不完整 |
| 网络分发   | 多渠道增长              | 单一渠道依赖     |
| 成本议价   | RLDC margin 稳定        | 分销成本率上升   |
| 使用场景   | 支付、结算、DeFi 多场景 | 只靠交易所余额   |
| 平台化收入 | Other revenue 占比上升  | 仍主要靠利息     |
