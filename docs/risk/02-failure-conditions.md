# 投资逻辑失效条件

## 结论

以下条件出现时，CRCL 研究框架必须从估值讨论切换到生存和信用讨论。

失效条件不是普通利空。
失效条件意味着原投资假设不再成立。

## 硬性失效条件

| 条件                                     | 原因               | 动作                   |
| ---------------------------------------- | ------------------ | ---------------------- |
| USDC 1:1 赎回信任受损                    | 稳定币信用基础破坏 | 停止所有平台化估值讨论 |
| Circle Reserve Fund 或储备披露异常       | 储备安全被质疑     | 优先确认赎回和资产质量 |
| USDC/USD 长时间折价                      | 市场不相信 1:1     | 进入信用事件处理       |
| 连续净赎回叠加市占率下降                 | 需求和竞争力双弱   | 下修收入和倍数         |
| 监管明确压制收益分享或第三方激励         | 分销经济性重写     | 重新建模 RLDC          |
| Other revenue 长期无法增长               | 平台化叙事失败     | 回到利差股估值         |
| Coinbase 或核心渠道垂直整合并削弱 Circle | 分销入口被控制     | 下调长期护城河         |

## 可恢复条件

| 失效信号           | 解除条件                           |
| ------------------ | ---------------------------------- |
| 短期净赎回         | USDC 30D change 转正，市占率稳定   |
| USDC 折价          | 二级市场恢复锚定，赎回通道正常     |
| 储备报告延迟       | 官方解释清楚，第三方数据无异常     |
| RLDC margin 下滑   | 管理层解释一次性因素，下一季度恢复 |
| Other revenue 放缓 | CPN / Arc 披露真实收入贡献         |

## 失效判断优先级

先信用，后收入。
先监管，后估值。
先现金流，后叙事。

如果信用层出问题，低估值不是买入理由。
如果监管重写分销模式，历史 RLDC margin 不能直接外推。

## Autoresearch 失效复核

失效条件必须先跑矩阵，再决定是否进入生存和信用讨论。

| 失效方向         | 必跑矩阵                         | 进入失效条件                         |
| ---------------- | -------------------------------- | ------------------------------------ |
| 信用失效         | 储备信用、赎回流动性             | USDC 折价、储备异常、赎回通道异常    |
| 现金流失效       | 财务十项复核                     | USDC 下滑、收益率下行、RLDC 同时恶化 |
| 分销失效         | 渠道议价矩阵、监管矩阵           | 第三方激励受限且渠道依赖上升         |
| 竞争失效         | 稳定币竞争矩阵、收益竞争矩阵     | 市占率下滑且资金迁移到替代品         |
| 平台化失效       | CPN 商业化复核、Arc 基础设施复核 | 使用量增长但 Other revenue 不增长    |

Other revenue 不披露可重复性时，只能判定平台化未验证。
Other revenue 长期不增长时，才进入平台化失效条件。
