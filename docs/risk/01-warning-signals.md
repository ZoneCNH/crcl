# 预警信号

## 结论

预警信号分为致命、严重、中等三类。

致命信号动摇信用。
严重信号动摇中期盈利或估值逻辑。
中等信号主要放大回撤，不单独破坏主框架。

## 预警清单

| 编号 | 预警信号                                       | 对应层级              | 严重程度 | 是否阻塞加仓 |
| ---- | ---------------------------------------------- | --------------------- | -------- | ------------ |
| 1    | Circle Reserve Fund 或储备报告异常 / 延迟      | 第 1 层 储备信用层    | 致命     | 是           |
| 2    | USDC 连续 2-4 周净赎回，流通量持续下降         | 第 2 层 利差现金流层  | 严重     | 是           |
| 3    | USDC 市占率持续下降，被 USDT 或银行稳定币侵蚀  | 第 10 层 竞争结构层   | 严重     | 是           |
| 4    | 分销成本率持续上升，未随规模扩大下降           | 第 4 层 成本议价层    | 严重     | 是           |
| 5    | Other revenue 占比停滞或下降，平台化未兑现     | 第 6 层 平台化期权层  | 严重     | 是           |
| 6    | 监管条款限制收益分享或让银行获得成本优势       | 第 5 层 监管牌照层    | 严重     | 是           |
| 7    | Coinbase / 主要合作伙伴垂直整合或议价权增强    | 第 3/4 层 分发与成本  | 严重     | 是           |
| 8    | 储备收益率明显下行，且 USDC 增速无法弥补       | 第 8 层 利率久期层    | 中等     | 视情况       |
| 9    | BTC / crypto beta 走弱，稳定币总供应量同步下降 | 第 11 层 宏观 beta 层 | 中等     | 视情况       |
| 10   | CRCL 高位放量跌破 20D/50D 均线，且基本面未改善 | 第 12 层 股票筹码层   | 中等     | 视情况       |
| 11   | short interest 与 put/call IV 同时快速上升     | 第 12 层 股票筹码层   | 中等     | 视情况       |
| 12   | 锁定期解禁、内部人大幅减持、增发临近           | 第 12 层 股票筹码层   | 中等     | 视情况       |
| 13   | Base USDC 增长但总 USDC 份额不升               | 第 3/10 层 渠道与竞争 | 严重     | 是           |
| 14   | USDe / RWA AUM 增长伴随 USDC 净赎回            | 第 10 层 竞争结构层   | 严重     | 是           |
| 15   | CPN TPV 增长但 Other revenue 不增长            | 第 6 层 平台化期权层  | 严重     | 是           |
| 16   | 银行稳定币或 tokenized deposits 获得企业采用   | 第 5/10 层 监管与竞争 | 中等     | 视情况       |

put/call 是看跌期权和看涨期权比例，白话就是市场保护性或看空需求强不强。

## 严重程度定义

| 严重程度 | 含义                         | 动作                     |
| -------- | ---------------------------- | ------------------------ |
| 致命     | 动摇 USDC 信用或 1:1 赎回    | 立即停止加仓，优先减风险 |
| 严重     | 威胁中期盈利或估值逻辑       | 暂停加仓，重新估值       |
| 中等     | 引发回撤或波动，但未破坏结构 | 提高止损纪律，等待确认   |

## 组合触发

以下组合比单一信号更危险：

1. 净赎回 + 市占率下降。
2. 降息预期 + USDC 不增长。
3. Base USDC 增长 + RLDC margin 下滑。
4. 监管限制激励 + Coinbase 分销依赖上升。
5. Other revenue 不增长 + CPN / Arc 只披露使用量。
6. 解禁临近 + 高 IV + 基本面降级。
7. Base USDC 增长 + USDC 市占率不升 + RLDC margin 下滑。
8. USDe / RWA AUM 增长 + USDC 净赎回 + DeFi USDC 借款走弱。
9. 银行稳定币企业采用 + CPN TPV 无收入贡献。

## Autoresearch 预警矩阵

预警信号必须从单点指标升级为矩阵判断。

| 矩阵         | 预警触发                             | 严重程度升级条件                         |
| ------------ | ------------------------------------ | ---------------------------------------- |
| 财务十项复核 | RLDC margin 低于 38%、成本解释不足   | 连续两个季度未修复                       |
| 监管矩阵     | yield workaround 或第三方激励受限    | 直接影响 Coinbase、钱包、交易所奖励      |
| 渠道议价矩阵 | Base 增长但总 USDC 份额不升          | 同时出现 RLDC margin 下滑                |
| 稳定币竞争矩阵 | USDT dominance 上升、PYUSD / FDUSD 迁移 | 同时出现 USDC 市占率 4 周下降            |
| 收益竞争矩阵 | USDe / RWA AUM 上升                  | 同时出现 USDC 净赎回和 DeFi 借款走弱     |
| 平台化矩阵   | CPN / Arc 使用量增长                 | Other revenue 不增长或收入可重复性不披露 |

yield workaround 是收益规避结构，白话就是不直接叫利息，但经济效果像给收益。
