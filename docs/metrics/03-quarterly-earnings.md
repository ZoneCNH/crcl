# 季度财报检查清单

## 结论

季度财报决定 CRCL 是继续按利差股定价，还是开始获得平台基础设施倍数。

先看收入质量，再看叙事。
先看平均 USDC，再看期末 USDC。

## 五项第一优先级

| 顺序 | 指标                               | 目的             |
| ---- | ---------------------------------- | ---------------- |
| 1    | Average USDC in circulation        | 判断收入基数     |
| 2    | Reserve return rate                | 判断利率压力     |
| 3    | Distribution and transaction costs | 判断分销成本     |
| 4    | RLDC margin                        | 判断规模收益留存 |
| 5    | Other revenue                      | 判断平台化验证   |

Average USDC in circulation 是平均流通量，白话就是季度内平均有多少 USDC 在外面。
期末值可能被短期波动影响，平均值更能解释储备收入。

## 阈值

| 指标                | 强化           | 中性               | 降级                 |
| ------------------- | -------------- | ------------------ | -------------------- |
| Average USDC        | 同比和环比增长 | 同比增长、环比平稳 | 环比下降且无解释     |
| Reserve return rate | 稳定或高于预期 | 小幅下降           | 大幅下降且规模未抵消 |
| RLDC margin         | 高于 40%       | 38-40%             | 低于 38%             |
| Other revenue share | 超过 10% → [平台化切换条件①/5 满足]，单条件不触发切换，参见 valuation/00 | 5-10% | 低于 5% 且无增长 |
| FY 指引             | 上修           | 维持               | 下修                 |

## 管理层问答

必须追踪以下问题：

1. CPN 的 TPV、客户数、收费模式和收入确认方式。
2. Arc 主网时间表、费用模型、企业客户试点。
3. Coinbase、Binance、其他渠道的分成变化。

5. USDC 增长来自交易、DeFi、支付还是企业结算。
6. 锁定期、二次发行、内部人减持节奏。

## Earnings Call 文字稿分析模板

财报电话会是财报数字背后的管理层解释层，必须与 SEC filing 结合使用，不能单独用于估值结论。

### 听电话会的优先级顺序

1. **量化信息**（出现即记录）：任何涉及数字的表述，包括 CPN TPV、Arc 客户数、产品 pipeline、指引更新。
2. **政策表述**：对 GENIUS Act、OCC 规则、分销协议变化的直接表态。
3. **管理层语气**：信心变化、措辞加强或软化，结合财报数字判断是否一致。
4. **分析师追问**：被追问的问题往往是机构关注的核心分歧。

### 电话会关键追踪点

| 追踪类别 | 具体问题 | 记录格式 |
| -------- | -------- | -------- |
| CPN 商业化 | 是否披露 TPV 数字？是否披露收费模式或收入？ | 文字原话 + 是否可验证 |
| Arc 主网 | 时间表有无更新？有无企业客户案例？ | 文字原话 + 日期对比 |
| 分销协议 | Coinbase 或 Binance 协议有无变化？分成比例是否调整？ | 文字原话 + 影响方向 |
| GENIUS Act | 管理层如何评估最终规则对 RLDC margin 的影响？ | 文字原话 + 偏乐观/谨慎 |
| USDC 增长来源 | 管理层是否披露交易、DeFi、支付、企业各占比？ | 文字原话 + 是否可验证 |
| 指引更新 | FY 指引是否变化？变化方向和原因？ | 文字原话 + 幅度 |

### 语气强度分级

管理层语气不能替代数字，但可以帮助判断 missing_info 的性质：

| 语气 | 含义 | 操作规则 |
| ---- | ---- | -------- |
| 强调具体数字、给时间表 | 有验证意图 | 记录原话，等财报或 8-K 确认 |
| 提使用量但不给商业化数字 | 仍处早期 | 列入 missing_info，不上调估值 |
| 回避分析师追问 | 可能有不确定性 | 列入 missing_info，保守处理 |
| 下修或调整措辞 | 框架假设可能需要更新 | 触发预期差框架检查 |

## Q1 2026 特别检查

Q1 2026 财报发布日期为 2026-05-11。

本次财报是 IPO 后早期公开市场验证节点。
优先判断市场是否高估平台化速度。

重点：

- Other revenue 是否接近 FY2026 1.5-1.7 亿美元年化节奏。
- RLDC margin 是否维持 38-40%。
- CPN 和 Arc 是否给出更硬的收入线索。
- 监管规则是否影响分销激励。
- 管理层是否上修或下修 USDC 增长假设。

## Q1 2026 盈利前预期基准

财报前锚定预期基准，用于对比实际结果，避免用结果反推叙事。

### 数量预期锚

| 指标 | 基准预期 | 来源依据 | 强化阈值 | 降级阈值 |
| ---- | -------- | -------- | -------- | -------- |
| 平均 USDC 流通量 | 760-800 亿美元 | Q4 2025 期末 753 亿，环比增势延续 | >800 亿 | <720 亿且无外部解释 |
| Reserve return rate | 3.6-3.8% | Q4 2025 实际 3.8%，降息预期 25-50 bps 已部分计入 | >3.8%（降息未如期） | <3.4%（降息超预期传导） |
| Reserve income | 6.8-7.4 亿美元 | 平均 USDC × reserve return rate 粗算 | >7.4 亿 | <6.5 亿 |
| RLDC | 2.9-3.2 亿美元 | RLDC margin 约 41% 按储备收入口径粗算 | >3.2 亿 | <2.7 亿 |
| RLDC margin（主，÷储备收入） | 39-41% | FY2025 全年约 41%，Q4 2025 约 42% | >42% | <38%（告警基准） |
| Other revenue | 3,500-4,000 万美元 | FY2025 全年 1.098 亿 ÷4 季度 ≈ 2,700 万，指引节奏 1.5-1.7 亿/年 ÷4 ≈ 3,750-4,250 万 | >4,500 万（领先指引） | <3,000 万（落后指引节奏） |
| Adjusted EBITDA | 1.3-1.6 亿美元 | FY2025 全年 5.822 亿 ÷4 粗算 | >1.6 亿 | <1.1 亿 |

平均 USDC 流通量的预期基准比期末值更重要。
期末值可能受月末铸造/赎回波动，不直接反映季度收入。

### 定性预期锚

| 判断项 | 基准预期 | 触发上修 | 触发下修 |
| ------ | -------- | -------- | -------- |
| FY2026 指引 | 维持 1.5-1.7 亿美元 Other revenue 指引 | 上修 | 下修超过 10% |
| CPN 进展 | 提及 TPV 增长，但不披露收入 | 披露收费模式或收入数字 | 仅提使用量，无商业化信号 |
| Arc 主网 | 仍处试点，主网延期风险存在 | 主网宣布落地 + 客户案例 | 延期或无新公告 |
| 监管口径 | GENIUS Act 未出最终规则，管理层表述谨慎乐观 | 最终规则对分销激励中性或有利 | 规则限制第三方激励或收益分享 |
| 分销成本 | 环比 Q4 持平或小幅上升 | 分销成本率下降 | 分销成本率上升超过 2 ppts |

### 预期差框架

财报实际结果与基准预期的对比规则：

```text
实际 > 强化阈值：该指标进入强化状态
基准范围内：该指标中性
实际 < 降级阈值（单项）：触发追问，不单独降级
实际 < 降级阈值（2 项以上）：触发降级检查，对照情景模型

RLDC margin < 38%（无论其他）：单项触发降级检查
信用层 C-TRIGGER-A/B/C 满足：立即执行信用层协议，独立于以上逻辑
```

单指标超预期不改变情景，多指标共振才触发框架升降级。
