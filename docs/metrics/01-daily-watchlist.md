# 每日监控清单

## 结论

每日只盯 P0，不做全量研究。

每日目标是回答一句话：今天有没有信号要求重估 CRCL 风险。

## 5 分钟版

| 顺序 | 指标                    | 来源                | 动作                     |
| ---- | ----------------------- | ------------------- | ------------------------ |
| 1    | USDC circulating supply | Circle Transparency | 记录当前值、7D、30D 变化 |
| 2    | Net mint / redeem       | Circle Transparency | 判断是否连续净赎回       |
| 3    | 3M T-Bill / SOFR        | Treasury / NY Fed   | 判断储备收益压力         |
| 4    | BTC / ETH               | 行情平台            | 判断 crypto beta         |
| 5    | CRCL price / volume     | 行情平台            | 判断股价是否和基本面背离 |

## 事件版

遇到以下事件，当天升级为 P0 复核：

1. Circle 储备报告延迟或异常。
2. GENIUS Act、OCC、Treasury、FinCEN、OFAC 发布新规则。
3. SEC、CFTC 对稳定币、交易所或钱包采取行动。
4. Coinbase、Binance、银行合作方出现影响 USDC 分销的公告。
5. CRCL 放量跌破关键均线，且基本面没有改善。

FinCEN 和 OFAC 是美国金融犯罪与制裁执行机构，白话就是反洗钱和制裁规则的关键监管方。

## 触发阈值

| 信号                       | 级别 | 动作                             |
| -------------------------- | ---- | -------------------------------- |
| USDC 连续 2-4 周净赎回     | P0   | 检查市占率、Curve 折价、储备报告 |
| USDC/USD 持续折价          | P0   | 停止平台化估值讨论，先看信用风险 |
| 储备报告异常或延迟         | P0   | 检查官方解释和赎回通道           |
| 监管限制第三方激励         | P0   | 重算分销经济性                   |
| CRCL 放量跌破 20D/50D 均线 | P0   | 检查是否有筹码或基本面原因       |

20D/50D 均线是 20 日和 50 日平均股价，白话就是短中期趋势线。

## 误判控制

单日价格波动不能改变研究结论。
单日 USDC 下滑也不能直接判断结构恶化。

只有当 P0 信号连续出现，或和监管、储备、赎回、分销成本同时共振，才进入降级。
