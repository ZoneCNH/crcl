# 决策输出模板

## 结论

所有 CRCL 决策输出必须先结论，再依据，再动作。

禁止只写观点。
必须把观点映射到指标、层级和下一步。

## 日常监控模板

```text
结论：今日维持观察。

依据：
1. USDC 30D change 仍为正。
2. 3M T-Bill 无明显下行。
3. 无监管 P0 事件。

动作：
不调整框架，等待周度市占率和转账量确认。

风险：
如果连续净赎回扩大，切换到 P0 风险复核。

下次复盘触发：
Circle Transparency 更新或监管公告。
```

## 禁止输出

以下输出不合格：

- “长期看好，短期震荡。”
- “监管利好合规稳定币。”
- “Arc 进展不错。”
- “股价回调可以关注。”

合格输出必须说明：

1. 哪个指标变了。
2. 哪个假设变了。
3. 哪个估值或风险动作变了。
4. 什么时候复盘。
